Impute missing data

Author

Pankaj Chejara

Published

April 7, 2024

The task is to impute missing data in the obesity dataset. Data imputation is a set of techniques that replaces missing data in a dataset with a value determined based on the used technique. For example, replacing all missing values with the mean value of a numeric attribute.

In the obesity dataset there are missing values in all the attributes except Gender and weightClass.

The dataset has both numeric and categorical attributes which have missing values. The type of data conditions the type of data imputation techqinues to use. For example, numeric attributes can be imputed using mean, median; categorical attributes can be imputed by category occurring frequently.

1 Numerical data imputation

1.1 Imputing Height

Height can be imputed using the mean imputation technique. The average height in the dataset is 1.70. However, looking at Figure 1, it can be seen that there is a difference between the heights of males and females. The average male height is 1.75 while the same for a female is 1.64.

Therefore, the missing values in the height attribute will be replaced with the mean value of height from the participant’s gender group.

Show the code 
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# load dataset
df_org = pd.read_csv('obesity.data.txt', sep='\t')
df = df_org.copy()
plt.figure(figsize=(6,4))

# plotting height distribution of male and female
sns.boxplot(df, x='Height', y='Gender')

plt.show()
Note

Data imputation approach used: Mean imputation based on gender.

Show the code 
# code-summary: show imputation code

# height data for male and female
male_height = df.loc[df['Gender'] == 'Male','Height']
female_height = df.loc[df['Gender'] == 'Female','Height']

# imputing height data
df['Height_clean'] = df['Height']
df.loc[(df.Gender=='Male') & (df['Height_clean'].isnull()),['Height_clean']] = male_height.mean()
df.loc[(df.Gender=='Female') & (df['Height_clean'].isnull()),['Height_clean']] = female_height.mean()

2 Categorical data imputation

There are seven categorical attribute which have missing values. These attributes are 1. FHO 2. FAVC 3. CAEC 4. SMOKE 5. SCC 6. CALC 7. MTRANS

Show the code 
cat_attrs = ['FHO','FAVC','CAEC',
            'SMOKE','SCC','CALC',
            'MTRANS']
Show the code 
# plotting distribution

for cat_att in cat_attrs:
    plt.figure()
    sns.countplot(df, x=cat_att, hue='Gender')
    if cat_att == 'MTRANS':
        plt.xticks(rotation=90)
    plt.show()
Tip

In all categorical attributes, the majority category remains the same in the male and female groups. Therefore, frequency-based imputer can be used, replacing the missing value with the most occuring category

Show the code 
for cat_att in cat_attrs:
    new_att = f'{cat_att}_clean'
    
    # replacing missing value with most occuring category
    df[new_att] = df[cat_att].fillna(df[cat_att].mode().iloc[0])

clean_attributes = []

3 Disadvantage of the used imputation techniques

The imputation techniques used above do not take into account dependence among attributes. Though the imputation technique considers Gender while imputing but similar approach can be employed to check other attributes as well.

In this direction, KNNImputer uses the K-Nearest Neighbor algorithm to impute missing values. Moreover, this technique has also been demonstrated to give good results independent of the used dataset (Jadhav, Pramod, and Ramanathan 2019).

References

Jadhav, Anil, Dhanya Pramod, and Krishnan Ramanathan. 2019. “Comparison of Performance of Data Imputation Methods for Numeric Dataset.” Applied Artificial Intelligence 33 (10): 913–33. https://doi.org/10.1080/08839514.2019.1637138.
